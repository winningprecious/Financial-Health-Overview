#  Financial Performance Dashboard  

###  Project Overview  
This Power BI dashboard provides a comprehensive view of *income, expenses, and savings performance* across different individuals and months.  
The goal was to track financial health over time, highlight spending patterns, and identify opportunities to improve savings rates.  

###  Tools Used  
- *Power BI* – for dashboard design and data visualization  
- *Excel* – for initial data cleaning and transformation  
- *PowerPoint* – for layout wireframing before building in Power BI  

---

###  Key Metrics  
- *Total Income:* $1.8M  
- *Total Expenses:* $1.3M  
- *Net Balance:* $538K  
- *Savings Rate:* 29.6%  

---

###  Insights  
-  *Income Growth:* Average monthly income stands at *$152K*, showing steady inflows throughout 2023–2024.  
-  *Expense Trend:* Average monthly expenses of *$107K*, but spikes in certain months show possible spending surges.  
-  *Savings Pattern:* Despite high expenses, users maintained an average *savings rate close to 30%*, signaling good financial control.  
-  *Top Earners:* Cherish recorded the highest income, followed closely by Precious and John.  
-  *Yearly Comparison:* 2024 saw a higher income-to-expense ratio (*60.19% vs 39.81%*) compared to 2023, indicating improved performance.  

---

###  Takeaway  
This dashboard demonstrates how simple visual analytics can reveal deep financial insights — helping individuals or small teams track progress, control spending, and plan better.
